## 学习建议

**windows系统中程序出现问题**.

1. 保存并且结束进程
2. 救命三键（Crtl+Alt+Delete）
3. 不要直接关机和reset

*养成做记录的习惯，尤其是发现问题的事后，把错误信息和引发的问题归档*

*看到好文章要自己留一份copy，同时定好题目，归档归类*

*使用者迁就于机器，开发者让机器迁就于人*

## 主机规划和磁盘分区

在linux服务器中，主存储器比CPU还高级，因为如果主存储器不够大，就会使用到硬盘内存置换空间：swap，会影响到设备的效能


各个硬件在Linux中的中文名，选择好所需要的硬件设备之后，揭晓来需要了解一下各个硬件在Linux当中所扮演的角色

在linux中，每一个设备都被当做一个文
件来对待，IDE接口的硬盘文件名就是`/dev/sd[a-d]`

在linux系统中，几乎所有的接口设备都在`/dev`接口中，随意会看到很多其他名字。

软盘驱动器：`/dev/fd[0-7]`
打印机`/dev/lp[0-15]`
鼠标：`/dev/input/mouse[0-15]`
或者`/dev/mouse`表示当前鼠标
